Cara membuka aplikasi :
1. Download dan unzip aplikasi
2. Buka file explorer tempat terdownload aplikasinya
3. Buka folder "app" kemudian copy path nya
4. Buka cmd, lalu ketikan "cd" dan paste path yang sudah di copy sebelumnya
6. Tekan enter
7. Ketikan "python app.py" dan tekan enter
8. Setelahnya akan muncul link http://localhost:5000 pada github
9. Copy link tersebut lalu paster pada pencarian anda
10. Selamat anda berhasil mengakses aplikasi Pendeteksi Penyakit Jantung berbasis website
